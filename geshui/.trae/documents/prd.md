## 1. 产品概述
个税计算器是一款帮助用户快速计算个人所得税的工具，用户输入税前工资、五险一金扣除比例等信息后，可即时得出应缴个税金额和税后工资。
- 主要用途：帮助工薪阶层快速计算每月应缴个税，了解工资结构
- 目标用户：需要计算个税的上班族、财务人员
- 产品价值：简单易用，计算准确，界面美观

## 2. 核心功能

### 2.1 功能模块
1. **计算器主页面**：输入表单、实时计算、结果展示

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 计算器主页面 | 输入表单 | 税前工资输入、五险一金比例设置、专项附加扣除设置、起征点设置 |
| 计算器主页面 | 计算逻辑 | 根据最新个税税率表计算应纳税额 |
| 计算器主页面 | 结果展示 | 显示应缴个税、税后工资、详细扣除明细 |
| 计算器主页面 | 税率说明 | 展示当前个税税率表供用户参考 |

## 3. 核心流程
用户进入页面 → 输入税前工资 → 设置五险一金扣除比例 → 输入专项附加扣除 → 系统实时计算并展示应缴个税和税后工资 → 查看详细扣除明细

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：透明淡蓝色 (#E3F2FD 半透明) 作为窗口背景
- 辅助色：蓝色 (#2196F3) 作为强调色
- 按钮样式：圆角按钮，悬停有阴影效果
- 字体：使用现代无衬线字体，清晰易读
- 布局风格：居中卡片式布局，玻璃态(Glassmorphism)效果
- 背景：使用淡蓝色渐变背景

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI Elements |
|-----------|-------------|-------------|
| 计算器主页面 | 输入表单 | 标签 + 数字输入框 + 滑块（比例调节）|
| 计算器主页面 | 结果展示 | 大号数字展示，分栏显示各项明细 |
| 计算器主页面 | 税率说明 | 折叠面板，点击展开查看税率表 |

### 4.3 响应性
- 桌面端优先，同时适配移动端
- 窗口最大宽度500px，移动端自适应屏幕宽度
- 触控优化：输入框和按钮尺寸适合手指点击

### 4.4 动画效果
- 页面加载时淡入效果
- 输入变化时数字平滑过渡
- 按钮悬停时缩放和阴影效果
- 结果数字更新时的滚动动画
