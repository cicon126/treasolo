## 1. 产品概述

打字大师（Dazi）是一款沉浸式打字训练小游戏，通过单词从屏幕顶部下落的机制，帮助用户在紧张刺激的游戏体验中提升打字速度与准确率。目标用户为希望提升英文打字技能的学生、程序员及英语学习者。

## 2. 核心功能

### 2.1 用户角色
无需用户系统，单机游戏，即开即玩。

### 2.2 功能模块
1. **主菜单页面**：游戏标题、模式选择、难度选择、开始按钮
2. **游戏页面**：单词下落区域、输入框、计分板、生命值、连击显示
3. **结算页面**：最终分数、统计信息（WPM、准确率、最大连击）、重新开始

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

玩家打开游戏 → 主菜单选择模式与难度 → 点击开始 → 单词从顶部下落 → 玩家在输入框中打字 → 匹配成功则单词消除并得分+粒子特效 → 输入错误扣生命值 → 单词落地扣生命值 → 生命值归零或时间到 → 结算页面展示成绩

## 4. 用户界面设计

### 4.1 设计风格
- **风格方向**：赛博朋克霓虹风格——深色背景配合霓虹发光效果，营造紧张刺激的游戏氛围
- **主色调**：深灰/近黑背景 (#0a0a0f)，霓虹青 (#00f5d4) 作为主强调色，霓虹粉 (#f72585) 作为警告/扣血色
- **辅助色**：霓虹黄 (#fee440) 用于连击高亮，霓虹紫 (#7b2ff7) 用于特效
- **字体**：标题使用 Orbitron（科幻感显示字体），正文/单词使用 JetBrains Mono（等宽编程字体，贴合打字训练主题）
- **按钮风格**：圆角按钮带霓虹发光边框，hover 时发光增强
- **布局风格**：全屏游戏区域，底部输入框，顶部状态栏，无传统导航

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主菜单页面 | 游戏标题区 | Orbitron 大标题带霓虹发光动画，扫描线效果背景 |
| 主菜单页面 | 模式选择 | 矩形按钮组，选中态带霓虹边框发光 |
| 主菜单页面 | 难度选择 | 三段式选择器，颜色区分（绿/黄/红） |
| 主菜单页面 | 开始按钮 | 大号圆角按钮，脉动发光动画 |
| 游戏页面 | 下落区域 | 深色背景带网格线，单词以 JetBrains Mono 显示，已匹配字母变青色高亮 |
| 游戏页面 | 输入框 | 底部居中，霓虹边框，打字时边框颜色跟随匹配状态变化 |
| 游戏页面 | 状态栏 | 左侧分数+连击，右侧生命心形+时间，半透明背景 |
| 游戏页面 | 特效反馈 | 消除粒子爆炸（彩色碎片飞散），连击数字放大弹出 |
| 结算页面 | 成绩展示 | 大号分数居中，统计项卡片式排列，渐入动画 |

### 4.3 响应式设计
- 桌面优先设计，游戏区域居中
- 最小支持 768px 宽度
- 移动端自适应：下落区域缩小，虚拟键盘弹出时调整布局

### 4.4 游戏机制详细设计
- **单词池**：按难度分级，简单（3-4字母）、普通（4-6字母）、困难（5-8字母）
- **下落速度**：简单 30px/s、普通 50px/s、困难 70px/s，随分数增加逐渐加速
- **生命值**：初始5颗心，拼错扣1心，单词落地扣1心
- **计分**：基础分=单词长度×10，连击加成=连击数×5
- **自动锁定**：输入第一个字母时自动锁定匹配的最近下落单词，输入完整个单词自动消除
- **WPM 计算**：(正确字符数/5) / 游戏时长(分钟)
- **准确率**：正确按键数 / 总按键数 × 100%
